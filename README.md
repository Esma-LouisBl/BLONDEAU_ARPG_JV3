/!\\ Il faut ouvrir le WBP\_UI avant de lancer le jeu, sinon le cooldown des sorts n'apparaît pas



Contrôles :



Le personnage se déplace en direction du clic du joueur

A/Z/E : Sort 1/2/3



Mécanique :



Le joueur utilise ses points de vie pour lancer les sorts



Sorts :



BloodWall → fait spawn un mur là où est le curseur

BloodSpike → fait spawn un pique là où est le curseur

Recover → si un élément spawné par le joueur est sous le curseur, il est détruit et le joueur regagne l'équivalent en vie
